Contextual Guide

Contextual Guide is a place for registering the context and the circumstances that form the setting for an event. Events can take place in the virtual and physical world. Each event receives a stable Uniform Resource Identifier (URI) that can be used for referencing purposes. Events can contain arbitrary metadata describing context and circumstances. Additionally, events can redirect the user to the original source of information. Once created, events are searchable by places and by keywords.

Simple and stable

Contextual Guide aims at remaining simple and stable.

Integrable

Contextual Guide can easily integrated with other online platforms.

Hackable

Contextual Guide is open source and aims at being maintained by a community users.
